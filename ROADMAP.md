# Roadmap

基于项目当前真实能力的开发规划。

## 当前状态 (2026-04-02)

**已完成：**
- 36因子评分引擎（3秒/300只）
- 4个Agent协作（Router/Analyst/Trader/Reporter）
- 模拟交易（建仓/调仓/风控止损/净值追踪）
- Telegram Bot（16个命令）
- 数据源（mootdx T+0实时 + BaoStock历史 + AKShare）
- 回测（2024 +46.2%, 2025 +37.3%）
- IC追踪（36因子20日基线，30个因子已自动调权）
- 行业对比（300只行业分类）
- 新闻情绪（CNBC + Google News）

**数据：** 365K条行情 / 300只 / 2020-2026.04

---

## Phase 1: 稳定运行 (1-2周)

目标：系统在真实交易日稳定跑满2周，积累数据。

- [ ] 连续10个交易日无人值守运行
- [ ] 每日自动数据更新（mootdx 15:10）
- [ ] 每日自动调仓 + 日报推送
- [ ] 积累20天净值数据 → 周报/月报可用
- [ ] 验证IC追踪的日度数据（非单次快照）
- [ ] 记录实际调仓决策和结果

**成功标准：** 连续运行14天不崩溃，日报不中断。

---

## Phase 2: 策略验证 (2-4周)

目标：用实盘数据验证策略，而非回测。

- [ ] 净值曲线跟踪（对比沪深300基准）
- [ ] 月度胜率统计（调仓后收益 vs 基准）
- [ ] IC追踪积累60天数据 → 权重调整有统计意义
- [ ] 因子有效性动态报告（哪些因子在失效/增强）
- [ ] 风控实战验证（是否真的能止损保护）
- [ ] 假设生成器优化（提高成功率 33% → 50%+）

**成功标准：** 模拟组合跑赢沪深300基准。

---

## Phase 3: 数据增强 (1-2个月)

目标：丰富数据维度，提升分析深度。

- [ ] AKShare接入：财务报表（资产负债表/利润表/现金流量表）
- [ ] 北向资金数据（AKShare `stock_hsgt_north_net_flow_in_em`）
- [ ] 融资融券数据
- [ ] 行业指数行情（申万行业分类）
- [ ] 宏观数据（CPI/PPI/PMI/M2）
- [ ] 个股公告/财报日历

**成功标准：** 基本面因子IC从+0.02提升到+0.05+。

---

## Phase 4: 策略深化 (2-3个月)

目标：从"评分Top10"进化到多策略组合。

- [ ] 行业轮动策略（行业景气度 → 行业Top3）
- [ ] 防御策略（市场弱势时自动降仓/转债券）
- [ ] 多策略组合（进攻+防御+平衡，按市场状态切换）
- [ ] 动态仓位管理（当前是满仓，改为根据信号强度调仓）
- [ ] 自定义股票池（不限沪深300，用户自定义关注列表）

**成功标准：** 回测年化保持20%+，最大回撤控制在15%以内。

---

## Phase 5: 工程化 (持续)

目标：让项目可复用、可扩展。

- [ ] 配置化（因子权重/策略参数全部YAML配置）
- [ ] 日志审计（每次决策记录到数据库）
- [ ] 单元测试覆盖（当前仅e2e测试）
- [ ] Docker化部署
- [ ] 多用户支持（每个用户独立持仓）
- [ ] Web Dashboard（净值曲线/持仓/历史决策可视化）

---

## 不做的

- 实盘交易（模拟盘阶段）
- 高频交易（日线级别）
- 付费数据源依赖（保持免费数据）
- 期权/期货/加密货币
